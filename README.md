## Notes:

To change file permissions for executables, do the following:

On Linux or macOS, run:

chmod +x entrypoint.sh
git add entrypoint.sh
git commit

On Windows, run:

git add --chmod=+x -- entrypoint.sh
git commit

And then push the changes back up.
